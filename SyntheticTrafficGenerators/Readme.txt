*** Launch the script from a command line as following:

        python ConcurrentTrafficRequests.py --frequency 60 --concurrent 20 --indelay 100

*** Where
        "frequency" represents the delay between two consecutive session bursts, in seconds
        "concurrent" represents the number of sessions launched as part of a burst
        "indelay" represenets the delay between two consecutive sessions in a burst, in miliseconds
